Velvet Astro - Unified Cyber Defense Platform

Version: 1.0.0 | Date: December 18, 2025

1. Executive Summary

Velvet Astro is a cutting-edge Cyber Defense SaaS platform designed to empower organizations with advanced security capabilities. It bridges the gap between automated threat detection and human operator readiness by combining real-time monitoring, machine learning-driven anomaly detection, and interactive security training.

2. System Architecture

  • Backend: FastAPI (Python 3.10+)
  • Frontend: Next.js 15 (React, TypeScript, Tailwind CSS)
  • Database: SQLite/PostgreSQL with SQLAlchemy & Alembic
  • Auth: OAuth2, JWT, Argon2
  • ML: Scikit-Learn for anomaly detection

3. Features & Modules

  • Identity & Access: Multi-tenancy, RBAC, Risk-based Auth.
  • Detection: Anomaly detection, Threat Intel, Deception (Honey Tokens).
  • Response: Incident management, Playbooks.
  • Training: Phishing simulations, Security awareness modules.
  • Visualization: Real-time Dashboard, Threat Map, Attack Feed.

5. Installation

Backend

cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload

Frontend

cd frontend
npm install
npm run dev

Docker

docker-compose up --build
